# snapshot-timemachine-rsnapshot

## Summary

rsnapshot backend for snapshot-timemachine

## Installing

You will need Emacs 24+, `make` and [Cask](https://github.com/cask/cask) to
build the project.

    cd snapshot-timemachine-rsnapshot
    make && make install
